# Arquitectura monolítica 

## Definicion
La arquitectura monolitica es la arquitectura donde el programa de software es unificado y la forma mas tradicional de realizar este codigo.

Cuando decimos monolito nos referimos a una strcutura grande y compacta

Esta estrcutura es sumamanete practica con proyectos pequeños o con principios de proyectos
